# Mod de Lector de Pantalla para DDLC Plus

Un mod que te permite disfrutar la experiencia completa de **Doki Doki Literature Club Plus** mediante lectores de pantalla.

---

## Características del Mod

* **Soporte Completo de Diálogos:** Todos los diálogos, narración y mensajes del sistema se envían a tu lector con identificación del hablante.
* **Navegación por Menús:** Desplazate por el menú principal, configuración, guardado/carga e historias secundarias.
* **Minijuego de Poesía:** Jugá con anuncios de selección de palabras y seguimiento de progreso.
* **Explorador de Archivos:** Interactuá con el sistema de archivos interno; los archivos se abren en el Bloc de Notas automáticamente.
* **Galería e Imágenes:** Descripciones completas para CGs, fondos, bocetos y contenido secreto.
* **Historias Secundarias:** Soporte total con navegación clara por capítulos.
* **Email y Música:** Acceso completo con anuncios de pistas y estados de reproducción.

## Guía de Instalación

### Paso 1: Instalar MelonLoader
Este mod requiere **MelonLoader** para funcionar.

1. Descargá el instalador desde [GitHub (última versión)](https://github.com/LavaGang/MelonLoader/releases/latest/).
2. Ejecutá `MelonLoader.Installer.exe`.
3. Hacé clic en **SELECT** y buscá la carpeta de instalación del juego.
4. Seleccioná el archivo `Doki Doki Literature Club Plus.exe`.
5. Hacé clic en **INSTALL**.
6. Iniciá el juego una vez para que se creen las carpetas y luego cerralo.

### Paso 2: Instalar el Mod
1. Extraé el ZIP del mod.
2. Copiá el archivo `DDLCScreenReaderMod.dll`.
3. Dirigite a la carpeta del juego y abrí la carpeta `Mods`.
4. Pegá el archivo ahí dentro.

### Paso 3: Configuración del Portapapeles
El mod envía el texto al portapapeles. Es necesario activar la lectura automática. Recomiendo el complemento [**Autoclip** para NVDA](https://nvda.es/2023/07/21/autoclip/).

## Atajos de Teclado

| Tecla | Función |
| :--- | :--- |
| **R** | Repite la última línea de diálogo. |
| **P** | Anuncia la pista actual (Reproductor de música). |
| **C** | Informa porcentaje de recolección de datos. |

## Problemas Conocidos

**Configuración de Escritorio:** En la aplicación de escritorio, las opciones de **Resolución** e **Idioma** podrían no anunciarse al enfocarlas. 
* **Solución:** Usá las flechas izquierda/derecha para cambiar los valores; el cambio sí será anunciado correctamente.
