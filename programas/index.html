Programas

Acá encontrarás algunos programas que… ¿Creé?

Bueno, es complicado. Las ideas son mías, pero el desarrollo está impulsado enteramente por IA y, en consecuencia, el código no me pertenece de ninguna manera.

Agarrá lo que quieras, usalo como quieras ah.

Cada programa incluye su propio manual; lo que hay en esta página es solo una breve descripción.

WinOCRScanner

Este es el hermano menor de PaddleOCRScanner. Hace lo mismo, pero de manera más rápida y, en algunos casos, menos precisa.

Tiene las mismas funciones que aquel, pero este usa el motor de OCR nativo de Windows. Está pensado para computadoras menos pudientes. Igual, sepan que mi favorito es el otro. Este es el hijo descuidado que se convierte en delincuente y roba carteras en los colectivos.

Podés descargar WinOCRScanner desde GitHub.

PaddleOCR Scanner

Sí, el nombre es genérico, ya sé. Este programa es una alternativa al OCR de Windows 10 y nació porque no podía leer los menús de una novela visual.

Hasta tiene escaneo dinámico y todo jajaj. Lo que sea de cada quién, me cebé con esto.

Podés Descargar PaddleOCR Scanner desde GitHub.

SpectrOCR

Este es un programa que agarra una ventana inaccesible con NVDA y la convierte en una página de navegador perfectamente manejable. Depende al completo de la IA de Google y, para ser honesto, no sé si quiero darle soporte, porque me ilusioné con Paddle. Pero no lo considero abandonado, todavía. No mientras me sea útil para algo.

Podés Descargar SpectrOCR desde GitHub.

Radio Browser

Este es un programa que permite escuchar todas las radios alojadas en el servicio de Radio Browser.

Podés Descargar Radio Browser desde GitHub.

Traductor de documentos

Este es un programa que traduce documentos y es compatible con los formatos más populares.

Aviso importante: Este fue mi primer proyecto y, lamentablemente, perdí el código fuente. Por este motivo, el programa no recibirá más actualizaciones; se quedará tal cual está como una pieza de museo personal.

Podés Descargar Traductor de Documentos desde Archive.
